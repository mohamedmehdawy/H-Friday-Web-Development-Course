server

جهاز متصل بشبكة الانترنت بقدر عن طريقة اشارك الملفات

client

المستخدم

VPN

هو عباره عن شبكة افتراضية بقدر عن طريقها تغير عنواني الموجود علي الانترنت

ip

عباره عن عنوان اي شخص الموجود علي الانترنت

domain

عباره عن اسم بيستخدم للوصول للسيرفر

TLD

بيعبر عن الجزء الاخير للدومين والي بيكون بيعبر عن التصنيف الي بينتمي ليه الدومين

URL

عباره عن الرابط الي بقدر عن طريقة الوصول للمصادر الموجوده علي السيرفر

browser

البرنامج الي بقدر عن طريقة وضع الرابط والوصول الي المصادر ببروتوكول HTTP

HTTP

البروتوكول الخاص بنقل الملفات

HTTPS

البروتوكول الخاص بنقل الملفات بشكل مشفر

protocol

الطريقة الي بقدر عن طريقها الوصول للمفات

SSH

البروتوكول المستخدم في التحكم بالسيرفر

FTP

البروتوكول المتسخدم في نقل وتعديل وحذف الملفات في السيرفر

HTML

اللغه المستخدمه في انشاء نصوص غير عادية ويتم قرائتها عن طريق بروتوكول HTTP

port

النافذه الي بيتم عن طريقها الوصول الي السيرفر

ISP

مزود خدمه الانترنت والي بيسمحلي بالاتصال بشبكة الانترنت
